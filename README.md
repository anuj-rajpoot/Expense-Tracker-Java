# Expense Tracker (Java)

A simple **console-based Expense Tracker** built in Java.  
This project allows users to **add, view, edit, delete, and summarize expenses**.  

---

## Features
- Add new expense (category, amount, date)
- View all expenses
- Edit existing expenses
- Delete expenses
- Show total expenses

---

## Technologies Used
- Java (Core Java, OOP)
- ArrayList for storing expenses
- Scanner for user input

---
